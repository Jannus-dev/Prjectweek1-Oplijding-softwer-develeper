Pseudocode: Pseudocode wordt gemaakt in Word, het beschrijft de werking van een programma die je gaat realiseren in het nederlands, zodat het voor iedereen te lezen is en zodat iedereen kan begrijpen hoe het programma werkt zonder programmeer kennis te hebben.
Flowchart: Vanuit de pseudocode maak je een flowchart van het programma, deze moet precies gelijk zijn aan je pseudocode. Je zorgt ervoor dat je programma werkend is binnen je flowchart.
Java: Je maakt vervolgens in java het programma wat weer gebaseerd is op je flowchart. Hierin kan je wel wat aanpassingen doen, omdat java meer functies heeft die je niet kan gebruiken in je flowchart.
